# OSINT Challenge: Accidents Happen

## Challenge Description

I was traveling and had a car accident with a bus. I managed to get a photo of the bus's license plate, but I didn't capture the full number. However, I remember seeing a company logo on it. Help me find the name of the company that owns this bus. Also find the ID of this bus that the company uses.

> [!IMPORTANT]  
> **Disclaimer:** This challenge description is fictional and for educational purposes only; no accidents ever happened.

## Objective
* Identify the **Company Name** that owns the bus.
* Identify the unique **Bus ID** (Internal ID) used by that company for this specific vehicle.

## Flag Format
The flag should be submitted in the following format:
`GIUCTF{Company_Name_Number}`

**Example:** `GIUCTF{Cool_Company_67}`
