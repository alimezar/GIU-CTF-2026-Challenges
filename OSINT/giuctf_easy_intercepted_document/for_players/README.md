# OSINT Challenge: Intercepted Document

## Challenge Description

Hello Agent, 

We intercepted a document that was sent to some underground hackers. We need you to find out who they are targeting urgently. Identifying the victim is our top priority to prevent a potential breach.

Once you have identified the target, we need you to find the specific password being discussed in their communications so we can inform the victim exactly which of their credentials has been compromised.

> [!IMPORTANT]  
> **Disclaimer:** This challenge description is fictional and for educational purposes only. All names and scenarios are part of a controlled CTF environment.

## Objectives
1.  **Identify the Victim:** Find the email address of the person or entity being targeted by the hackers.
2.  **Recover the Password:** Find the leaked password associated with the victim's account.

## Flag Formats

### Part 1: The Victim
The first flag is the email address of the target.
**Format:** `GIUCTF{example@email.com}`

### Part 2: The Password
The second flag is the leaked password found in the intercepted data.
**Format:** `GIUCTF{password}`
