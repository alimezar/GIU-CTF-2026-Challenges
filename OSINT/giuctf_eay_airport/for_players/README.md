# OSINT Challenge: Airport

## Challenge Description

A suspect currently on the move has posted a single image to their social media. However, they are professionals and have heavily blurred key parts of the photo. We need you to use your investigative skills to bypass these obfuscations and identify their passport number.

> [!IMPORTANT]  
> **Disclaimer:** This challenge description is fictional and for educational purposes only. Any resemblance to real persons or actual documents is purely coincidental.

## Objective
* Identify the **Passport Number** of the suspect based on the provided social media image and surrounding context.

## Flag Format
The flag should be submitted in the following format:
`GIUCTF{A12345678}`
