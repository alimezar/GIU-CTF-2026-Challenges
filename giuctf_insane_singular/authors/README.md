# Singular Intentions

**Difficulty:** Insane  
**Flag:** `GIUCTF{gr0ebn3r_b4s1s+s4g3!}`

## Challenge description
Three ciphertexts were torn out of the same machine, one modulus was left behind, and someone leaked a single algebraic relation that should never have survived. The exponent is missing, the messages are split, and the arithmetic looks just ugly enough to waste your night. It isn't. Can you collapse the system?

## Player handout
- `chall.sage`
- `output.txt`

## Intended solve path
1. Read `chall.sage` and notice that three plaintext chunks are RSA-encrypted under the same modulus with a secret prime public exponent smaller than `2^10`.
2. Notice the extra leak `hint = m1 + m2 + m3`, which gives one linear relation between the three unknown message chunks.
3. Model the system over `Zmod(N)`:
   - `m1 + m2 + m3 = hint`
   - `m1^e = c1 mod N`
   - `m2^e = c2 mod N`
   - `m3^e = c3 mod N`
4. Brute-force the unknown exponent over the prime values below `2^10`.
5. For each candidate exponent, use SageMath to compute a Gröbner basis for the ideal generated by the four equations.
6. When the basis reduces to three linear polynomials, recover the three message chunks and concatenate them into the flag.

## Solver
Run with SageMath:
```bash
sage solver.sage
```
